SharpTodo
=========

A minimalistic example of C# DDD architecture, consisting in a todo list, available via Console gui and REST interface.

Work in progress...

Copyright (C) [Massimo Barbieri](http://www.massimobarbieri.it) 

## Environment

* .NET Framework / C#
* Entity Framework
* Web API
* Ninject
* Moq
* NUnit
